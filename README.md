## Comparativo de Frameworks CSS: Implementação com Bootstrap

Este projeto faz parte do meu Trabalho de Conclusão de Curso (TCC) em Sistemas para Internet no IFSP, onde realizei uma análise prática sobre a usabilidade, padronização e produtividade de diferentes frameworks CSS.

## Sobre o Projeto
O objetivo desta aplicação foi validar a eficiência do **Bootstrap** na criação de interfaces consistentes e funcionais. Através da abordagem baseada em componentes, explorei a velocidade de desenvolvimento e a facilidade de organização oferecida pelo framework, utilizando sua vasta documentação e elementos pré-configurados.

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica.
* **Bootstrap 5.3**: Estilização baseada em componentes e sistema de grid responsivo.
* **JavaScript (Bootstrap Bundle)**: Utilizado para gerenciar a interatividade e as transições nativas de componentes (como o menu) de forma integrada.
* **GitHub Pages**: Hospedagem da aplicação.
